⚠️ Important Development Guidelines: Please read these instructions before working with this mockup.
  • Mockup-Only Development: We are creating mockups exclusively. No real model data will be saved to the database during this phase.
  • No Database Persistence: We will not create controller logic that saves, updates, or deletes real data from the database.
  • Sample Data Only: Controller methods will render sample data to ERB files for visualization purposes only.
  • Static Mockups: All forms, tables, and interactive elements are for demonstration and should display predefined sample data.
  • No Backend Logic: Focus is on frontend presentation, not backend functionality or business logic implementation.
